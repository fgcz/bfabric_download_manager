  # B-Fabric Download Manager

  This repository serves as a distribution point for the B-Fabric Download Manager JAR files.

  ## Download

  The latest release can be downloaded directly from the
  [Releases](https://github.com/fgcz/bfabric_download_manager/releases) page.

  Download the JAR file:
  https://github.com/fgcz/bfabric_download_manager/releases/download/v5.0.3/downloadmanager.jar

  ## Development

  Active development happens on our GitLab repository:
  https://gitlab.bfabric.org/bfabric/bfabric-download-manager

  This GitHub repository is used solely for public distribution of compiled JAR files.
